# Autocompleter

- Version: 1.0
- Author: Marcin Konicki (http://ahwayakchih.neoni.net)
- Build Date: 01 May 2011
- Requirements: Symphony version 2.2 or later.


## Overview

This extension allows developers to add autocomplete functionality to selected textarea and input fields.


## Installation

1. Upload the 'autocompleter' folder in this archive to your Symphony 'extensions' folder.
2. Enable it by selecting the "Autocompleter", choose Enable from the with-selected menu, then click Apply.
3. You can now configure autocomplete functionality for each textarea and input field in section.
